### การทดสอบสคริปต์:

1. **ให้สิทธิ์การรันสคริปต์**: 
	``chmod +x myconfig-manager.sh``
2. **เรียกใช้งานสคริปต์**:
	``~/myconfig/myconfig-manager.sh help``
	or
	``./myconfig-manager.sh help``
